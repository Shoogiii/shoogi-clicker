============================================================
          SHOOGI CLICKER v2.0 - USER MANUAL
============================================================

Shoogi Clicker is a high-performance Auto Clicker application 
featuring a modern UI and advanced customization options.

------------------------------------------------------------
[ KEY FEATURES ]
------------------------------------------------------------
* Fast Clicking: Precise interval settings in milliseconds (ms).
* Anti-Detection: Adds random delays (jitter) to mimic human behavior.
* Global Hotkey: Start/Stop instantly using F6 (Default).
* Tray Support: Minimizes to the system tray to run in the background.
* Profile System: Save and load different settings for various games.
* UI Design: Modern "Cyber Violet" dark theme.

------------------------------------------------------------
[ HOW TO RUN? ]
------------------------------------------------------------
1. Launch the shoogi_clicker_v2.exe file.
2. Set your desired click interval (ms) on the main screen.
3. Click the "START" button or press F6 on your keyboard.
4. To stop, simply press F6 again.

------------------------------------------------------------
[ ADVANCED SETTINGS ]
------------------------------------------------------------
- Random Jitter: When enabled, clicks won't happen at the exact 
  same speed every time, helping bypass anti-cheat systems.
- Always on Top: Keeps the application window visible over 
  other windows.
- Statistics: Track your total click count and session data 
  directly from the dashboard.

------------------------------------------------------------
[ SAFETY NOTE ]
------------------------------------------------------------
If you cannot stop the clicker for any reason, quickly move 
your mouse cursor to the TOP-LEFT corner (0,0) of your screen. 
This is a built-in "Fail-Safe" that will force the program 
to stop immediately.
